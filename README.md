TSSCCjs
====
Ctrl-C MML Player Chrome extension

##FEATURES
 - Ctrl-C(Command-C) to play interface
 - play sound with TSS javascript sound driver
 - translate TSSCP MML to TSS MML
 - support #A-#Z macro
 - support #FM macro

##BUILD

download dependencies
    make init

build
    make build

test
    make test
